Codea IT - M.E.N challenge

Welcome to the Mongo + Express + Node challenge! We are working on a blog backend in which users can register, login and create/read posts.

Your api base url is http://localhost:3000/api/

Please read the README.md carefully before starting.

Ready to start? Here we go.

1. Authorization

Someone discovers there’s an issue with authorization. Any user can create a post on behalf of another user! Are you able to fix it?

2. Get posts

The existing endpoint GET http://localhost:3000/posts brings all the existing posts. Implement the following functionalities:

  • 2.a) Get posts by author
  • 2.b) Get posts by keyword

3. Comments

Many users are requesting to make comments on the posts.

Each comment should have an author and the content itself.

  • 3.a) Add a new comment to a post
  • 3.b) Include comments on Get post by id

4. Tests

We need unit tests for either points 2 or 3 to have better coverage!

5. [Optionals] Bonus Track.

  • 5.a) The frontend team is asking for api documentation. They’ve seen swagger as a good choice. Is it already implemented?
  • 5.b) Infrastructure asks to run the api in a docker container, pointing to a minified build.
